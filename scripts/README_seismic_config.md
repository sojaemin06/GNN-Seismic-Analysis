# 내진성능평가 설정 파일 가이드 (scripts/seismic_design_config.json)

이 문서는 `seismic_design_config.json` 파일의 각 설정 항목에 대한 설명과 입력 가능한 값들을 `KDS 41 17 00 : 2022` 및 `기존 시설물(건축물) 내진성능 평가요령(2021)`을 기준으로 안내합니다.

설계자는 이 JSON 파일을 수정하여 내진성능평가 대상 구조물의 대지 조건, 지진 구역, 목표 성능 수준(내진등급)을 변경할 수 있습니다.

---

## 1. site_parameters (지반 및 지역 설정)

### `site_class` (지반 종류)
건물이 위치한 대지의 지반 특성을 나타냅니다. (`KDS 41 17 00` 표 4.1-1 참조)

| 값 | 명칭 | 설명 (기반암 깊이 및 전단파속도 기준) |
| :--- | :--- | :--- |
| **"S1"** | 암반 지반 | 기반암 깊이가 3m 미만인 경우 |
| **"S2"** | 얕고 단단한 지반 | 기반암 깊이 3~20m, 토층 평균 전단파속도 > 260m/s |
| **"S3"** | 얕고 연약한 지반 | 기반암 깊이 3~20m, 토층 평균 전단파속도 ≤ 260m/s |
| **"S4"** | 깊고 단단한 지반 | 기반암 깊이 > 20m, 토층 평균 전단파속도 > 180m/s |
| **"S5"** | 깊고 연약한 지반 | 기반암 깊이 > 20m, 토층 평균 전단파속도 ≤ 180m/s (또는 분류 데이터 부족 시 기본값) |

### `Z` (지진구역계수) **[New]**
해당 지역의 지진구역계수를 입력합니다. (`내진성능 평가요령` 표 2.2.1 참조)
이 값에 재현주기별 위험도계수(I)를 곱하여 유효지반가속도(S)를 산정합니다.

| 지진구역 | 주요 행정구역 | 지진구역계수 (Z) |
| :--- | :--- | :--- |
| **I** | 서울, 인천, 대전, 부산, 대구, 울산, 광주, 세종, 경기, 충북, 충남, 경북, 경남, 전북, 전남, 강원 남부 | **0.11g** |
| **II** | 강원 북부, 제주 | **0.07g** |

### `importance_class` (내진등급)
건축물의 용도 및 중요도에 따른 등급입니다. **이 값과 계산된 설계스펙트럼가속도($S_{DS}, S_{D1}$)를 바탕으로 내진설계범주(A, B, C, D)가 자동으로 결정되며, 평가할 성능목표 세트도 자동으로 설정됩니다.**

| 값 | 설명 |
| :--- | :--- |
| **"Teuk"** | **특등급** (중요도계수 1.5 적용 대상, 지진 후 즉시 기능수행 필요 시설 등) |
| **"I"** | **1등급** (중요도계수 1.2 적용 대상, 대형 건축물 등) |
| **"II"** | **2등급** (중요도계수 1.0 적용 대상, 일반 건축물) |

---

## 2. 성능목표 자동 설정 (참고)

사용자가 `importance_class`를 선택하면, 스크립트는 내부적으로 다음 기준에 따라 성능목표를 자동 설정합니다. (`내진성능 평가요령` 표 2.2.2 위험도계수 적용)

| 내진등급 | 평가 항목 (재현주기) | 위험도계수 (I) | 유효지반가속도 (S) | 허용 층간변위비 |
| :--- | :--- | :--- | :--- | :--- |
| **특등급** | 2400년 (붕괴방지/인명보호) | 2.0 | Z × 2.0 | 2.0% (인명) |
| | 1000년 (기능수행) | 1.4 | Z × 1.4 | 0.7% |
| **1등급** | 2400년 (붕괴방지) | 2.0 | Z × 2.0 | 3.0% |
| | 1400년 (인명보호) | 1.6 | Z × 1.6 | 2.0% |
| | 100년 (기능수행) | 0.57 | Z × 0.57 | 0.7% |
| **2등급** | 2400년 (붕괴방지) | 2.0 | Z × 2.0 | 3.0% |
| | 1000년 (인명보호) | 1.4 | Z × 1.4 | 2.0% |
| | 50년 (기능수행) | 0.4 | Z × 0.4 | 0.7% |

## 3. 설정 파일 작성 예시

```json
{
    "project_name": "Project Alpha",
    "description": "Seoul Area Evaluation (Zone I)",
    "site_parameters": {
        "site_class": "S4",      // 서울 지역 일반적인 토사 지반
        "Z": 0.11,               // 지진구역 I
        "importance_class": "I"  // 내진 1등급 건물
    }
}
```
