# 개발 계획 (Development Plan)

## Phase 1: 데이터셋 구축 (완료)
- [x] OpenSeesPy 기반 구조 해석 엔진 구축 (`src/core/`)
- [x] 데이터 생성 스크립트 및 검증 로직 구현 (`scripts/generate_dataset.py`)
- [x] 그래프 데이터 변환 로직 구현 (`src/data/`)

## Phase 2: GNN 모델 개발 (현재 진행 중)
- **목표:** 구조 정보와 해석 방향을 입력받아 푸쉬오버 곡선을 예측하는 베이스라인 모델 확보.
- **주요 마일스톤:**
    1. GNN 모델 아키텍처 설계 및 구현 (Spec 02 기반)
    2. 학습 파이프라인(Train/Valid/Test) 구축
    3. 초기 모델 학습 및 수렴 확인 (Overfitting 테스트)

## Phase 3: 성능 고도화 및 평가
- **목표:** SCI 논문 수준의 예측 정확도 달성.
- **주요 마일스톤:**
    1. 하이퍼파라미터 튜닝
    2. 역량스펙트럼법(CSM) 연동 성능점 평가
    3. 오차 분석 및 시각화

## Phase 4: 논문 작성 지원
- **목표:** 실험 결과의 정량적/정성적 분석 자료 생성.
